# تحديث تطبيق مدى الإنسان

هذا التحديث يحوّل التطبيق من واجهة أولية إلى تطبيق أكثر اكتمالًا مرتبط بواجهات API الخاصة بالموقع.

## ما تم إصلاحه

- إضافة صفحة رئيسية حقيقية.
- إضافة شاشة أقسام وتصنيف المقالات حسب القسم.
- إضافة بحث في المقالات عبر `GET /api/v1/articles?q=`.
- إضافة قراءة آمنة للـ JSON حتى لا ينهار التطبيق إذا رجع الخادم HTML أو صفحة 404.
- إصلاح رسالة الخطأ الخاصة بتسجيل دخول الأدمن عندما يكون مسار API غير منشور.
- إضافة Dashboard أدمن مبسط.
- ضبط GitHub Actions لبناء APK release خفيف بدل debug.
- رفع artifact باسم `mada-alensan-arm64-release-apk`.

## مهم عن رابط login

فتح هذا الرابط في المتصفح:

```txt
https://mada-alensan.vercel.app/api/v1/auth/login
```

هو طلب GET، لذلك الطبيعي أن يرجع JSON بحالة 405 ويقول إن المطلوب POST.

تسجيل الدخول الحقيقي في التطبيق يستخدم POST مع البريد وكلمة المرور.

## بناء APK

من GitHub Actions:

```txt
Actions → Build Android APK → Run workflow
```

سيتم بناء:

```txt
app-arm64-v8a-release.apk
```

وهذا أخف من debug APK.
