# تقرير تنظيف وتجهيز النشر

تم تجهيز هذه النسخة لتكون أخف وأنظف للنشر.

## ما تم تعديله

- تحديث `.gitignore` لمنع رفع `node_modules` ومخرجات البناء مثل `.next` وملفات البيئة الحقيقية.
- تنظيف ملفات البيئة المساعدة من القيم الحساسة الحقيقية والإبقاء على أسماء المتغيرات فقط.
- إصلاح تحقق جلسة تسجيل الدخول بحيث لا يفشل عند استقبال توكن مشوّه، وإجبار `AUTH_SECRET` في الإنتاج.
- توحيد دعم رفع SVG بين مسار الرفع وطبقة التخزين حتى لا تسمح الواجهة بنوع ملف ترفضه طبقة التخزين.
- إزالة مخرجات البناء والكاش من النسخة النهائية المضغوطة.

## أوامر النشر المقترحة

```bash
npm ci
npm run build
```

ثم على Vercel تأكد من وجود المتغيرات الأساسية: `DATABASE_URL`, `DIRECT_URL`, `AUTH_SECRET`, `ADMIN_EMAIL`, `ADMIN_PASSWORD`, ومتغيرات Supabase/Telegram حسب الميزات المفعلة.
